## 📝 Descripción

Este repositorio contiene el backend de **REACT GAMES**, una API REST diseñada para reemplazar por completo el uso de `localStorage` en nuestra aplicación de juegos de mesa. Permite persistir los datos de forma real mediante una base de datos relacional y proporciona operaciones CRUD completas para la gestión de la ludoteca digital.

## 🚀 Tecnologías utilizadas

- **Node.js** & **Express** (Servidor web)
- **Prisma ORM** (Modelado y consultas)
- **PostgreSQL** (Base de datos alojada en Neon.tech)
- **Cors** & **Dotenv** (Configuración y seguridad)
- **Nodemon** (Entorno de desarrollo)
